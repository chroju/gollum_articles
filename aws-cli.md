profile
----

実行アカウントをプロファイルで管理している。現在のプロファイルを確認するには以下。

```bash
$ aws configure list
      Name                    Value             Type    Location
      ----                    -----             ----    --------
   profile                <not set>             None    None
access_key     ****************RE2A              env
secret_key     ****************9DO2              env
    region           ap-northeast-1              env    AWS_DEFAULT_REGION
```
